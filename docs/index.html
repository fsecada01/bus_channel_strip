Professional Bus Channel Strip

A 6-module serial bus processor built with NIH-Plug and Airwindows-derived DSP algorithms in Rust. Designed for mastering-grade signal processing on mix buses, stems, and instrument groups. Every module is reorderable, bypassable, and fully automatable.

VST3 CLAP Rust + NIH-Plug vizia GUI Airwindows DSP Lock-Free Audio Thread
API5500 EQ → ButterComp2 → Pultec EQ → Dynamic EQ → Transformer → Punch

The default signal chain models a classic studio workflow: corrective EQ first (API5500), then dynamic glue compression (ButterComp2), tonal shaping (Pultec), frequency-dependent dynamics (Dynamic EQ), harmonic coloration (Transformer), and finally transparent peak limiting with transient restoration (Punch). All six modules can be reordered via the drag-to-swap handles in the GUI.

Design Philosophy

Each module targets a specific stage of the professional bus processing workflow. Used together in a single plugin instance, they replace a typical hardware or multi-plugin mastering chain — with zero inter-plugin latency, consistent gain staging throughout, and fully automatable parameters for dynamic processing changes.

Module Reference

All modules feature an individual bypass switch. Parameters are fully automatable via VST3 and CLAP protocols.

EQ
API5500 EQ
5-band semi-parametric equalizer with API console character

Models the frequency response character of the API 5500 Dual Channel EQ — a classic console equalizer known for its musical, punchy sound. The API 5500 was a staple on New York hit records through the 1970s–90s for its ability to add energy without sounding harsh. Semi-parametric design means frequency selection is stepped (classic API behavior), while gain is continuously variable. The circuit topology imparts a subtle harmonic character even at 0 dB boost/cut positions.

ControlRangeDescription
High-Pass Filter 20 – 600 Hz 12 dB/oct Butterworth HPF. Use at 30–60 Hz to clear sub rumble on mix bus without affecting kick energy.
Low Shelf ±15 dB, 30–200 Hz Low-frequency shelving boost/cut. API-character shelf adds body without boom when boosted moderately (+3–4 dB).
Low-Mid Bell ±15 dB, 100 Hz – 3 kHz, Q 0.5–5 Parametric peaking band. Primary tool for clearing mud (200–400 Hz cuts) or adding warmth (100–200 Hz boosts).
High-Mid Bell ±15 dB, 500 Hz – 8 kHz, Q 0.5–5 Presence and upper-mid control. +2–4 dB at 2–4 kHz adds punch and intelligibility to mix.
High Shelf ±15 dB, 2–20 kHz Air and brightness control. API-style high shelf sounds smooth; +2–4 dB at 10–16 kHz adds sparkle.
Classic Technique: API Smile Curve

Low shelf +3 dB at 80 Hz • Low-mid cut −2 dB at 300–400 Hz • High shelf +3 dB at 10 kHz. This classic console move adds energy and excitement to a mix without increasing perceived loudness excessively. The low-mid dip creates separation between bass and kick, while the dual boosts flatter the frequency extremes.

CMP
ButterComp2
Airwindows bipolar interleaved compression system

Chris Johnson's ButterComp2 is described as “the single richest, lushest glue compressor.” Rather than using a traditional VCA or optical circuit model, ButterComp2 implements four independent compressors per stereo channel in a bipolar, interleaved configuration. This creates a complex, harmonically rich gain reduction character that feels more like analog hardware than most plugin compressors. The result is a compression that “breathes” with the music rather than squashing it.

ControlRangeDescription
Compress 0.0 – 1.0 Overall compression depth. 0 = unity, 1 = maximum. Values above 0.6 become aggressive. Sweet spot: 0.3–0.55 for glue, 0.6–0.8 for NY parallel.
Output 0.0 – 1.0 Post-compression output level. Compensates for gain reduction. Start at 0.75–0.85 and trim to match bypass level.
Dry/Wet 0.0 – 1.0 Parallel mix of dry signal with compressed signal. This is the built-in NY-parallel blend control. 0.3–0.5 = subtle glue, 0.7+ = heavy compression character.
NY-Style Parallel Compression (The Classic Trick)

The “New York compression” technique blends heavily-compressed signal under the dry signal to bring up room ambience and micro-detail without affecting punch and dynamics. With ButterComp2: set Compress to 0.72–0.82, Output to 0.88–0.92, Dry/Wet to 0.28–0.42. The compressed signal adds density and sustain while the dry signal preserves attack transients. Start at Dry/Wet 0.3 and increase until you hear the bottom of the mix “fill in.”

PEQ
Pultec EQ
EQP-1A style passive tube equalizer with simultaneous boost/cut

The Pultec EQP-1A is one of the most revered equalizers in recording history. Its passive tube circuit allows a unique behavior: the low-frequency boost and cut controls can be applied simultaneously, creating a characteristic low-end curve with a tight, controlled peak followed by an attenuated shelf. Original hardware units from the 1950s sell for tens of thousands of dollars. This module models that behavior including the tube saturation coloration.

ControlRangeDescription
LF Boost Freq 20, 30, 60, 100 Hz Center frequency for low-frequency boost. 60 Hz is the classic kick and bass fundamental frequency.
LF Boost 0 – 1.0 (0–8 dB) Low shelf boost amount. Adds warmth, weight, and sub energy. Quadratic mapping for gentle resolution at low values.
LF Cut 0 – 1.0 (0 to −6 dB) Simultaneous low-frequency cut. Applied at a lower frequency than the boost, creating the iconic Pultec low-end shape.
HF Boost Freq 5, 8, 10, 12, 15, 20 kHz High-frequency boost center. 10–12 kHz for presence/air; 5–8 kHz for brightness in vocal mix.
HF Boost 0 – 1.0 (0–10 dB) High peaking boost. Adds presence and air. The Pultec high boost is famous for being non-fatiguing even at high amounts.
HF Boost BW 0 – 1.0 (Q 0.6–2.0) Bandwidth of HF boost. Low = wide shelf-like; high = narrow bell.
HF Cut Freq 5, 10, 20 kHz High-frequency shelving cut to control harshness or tape-style rolloff.
HF Cut 0 – 1.0 (0 to −8 dB) Amount of HF shelf cut.
Tube Drive 0 – 1.0 Tube saturation character. Gentle tanh-based soft saturation modeling. Low values (0.1–0.3) add analog warmth without obvious distortion.
The Pultec Trick (Simultaneous Boost + Cut)

Set LF Boost Freq to 60 Hz, LF Boost to 0.55, and LF Cut to 0.40 simultaneously. The boost adds sub energy at 60 Hz; the cut attenuates the low-mid bloom slightly below that frequency. Result: a tight, controlled low end with real sub presence and no mud — the hallmark Pultec sound that decades of engineers have tried to replicate. The two controls interact to form an EQ curve no single filter can produce.

DEQ
Dynamic EQ
4-band frequency-dependent dynamic processing with spectral analyzer

Dynamic EQ combines the precision of parametric EQ with the reactivity of dynamics processing. Each band operates as an envelope-follower-driven peaking filter: it only applies equalization when the sidechain level crosses the threshold, proportional to the overage and ratio. Three modes give it the flexibility to function as a multiband compressor (Compress Down), multiband expander (Expand Up), or gating notch filter (Gate).

Per-Band Controls (x4)

ControlRangeDescription
Mode Compress / Expand / Gate Compress Down reduces gain when signal exceeds threshold. Expand Up increases gain above threshold. Gate attenuates below threshold.
Detector Freq 20 Hz – 20 kHz Sidechain filter center frequency. The envelope follower measures level at this frequency, allowing frequency-specific triggering independent of the EQ band center.
EQ Freq 20 Hz – 20 kHz Center frequency of the dynamic peaking filter applied to the signal.
Q 0.1 – 10.0 Bandwidth of the dynamic EQ bell. Lower Q = wider, gentler; higher Q = narrower, more surgical.
Threshold −60 – 0 dBFS Level at which the band begins engaging. Below threshold: no EQ. Above threshold: gain change proportional to ratio.
Ratio 1:1 – 20:1 Compression or expansion ratio. 2:1 is gentle; 4:1 is assertive; 10:1+ is limiter territory.
Attack 0.1 – 200 ms Time constant for envelope follower attack. Fast (1–5 ms) catches transients; slow (20–100 ms) allows transients through.
Release 10 – 2000 ms Recovery time. Too fast = pumping; too slow = over-damping. Program-dependent behavior: 80–300 ms is typical.
Makeup Gain −12 – +12 dB Static gain applied after dynamic processing. Compensate for gain reduction or add frequency-specific level.
Solo On/Off Band isolation mode: routes only this band's frequency range through a bandpass filter for monitoring. Multiple bands can be soloed simultaneously.
Mastering De-Essing via Dynamic EQ

Traditional de-essers can sound abrupt. Use Band 3 or 4 in Compress Down mode: Detector Freq 7–8 kHz • EQ Freq 7 kHz • Q 1.5–2.0 • Threshold −14 to −10 dBFS • Ratio 2:1–3:1 • Attack 3–8 ms • Release 50–100 ms. This compresses only when the sibilant content exceeds threshold, preserving the natural HF energy of cymbals and acoustic instruments between sibilant events.

XFM
Transformer
Vintage transformer coloration with 4 hardware models

Transformer coloration is one of the most subtle but powerful aspects of analog hardware character. Input and output transformers in classic consoles impart low-frequency weight (core saturation), high-frequency softening or sparkle, and harmonic content that gives recordings a three-dimensional quality. This module models input and output transformer stages with four distinct models.

ControlRangeDescription
Model Vintage / Modern / British / American Selects the frequency response character and harmonic signature. Changes filter corner frequencies and saturation curve shapes.
Input Drive 0.0 – 1.0 Drive level into the input transformer stage. Increases harmonic content. Models impedance loading effect.
Input Saturation 0.0 – 1.0 Input transformer core saturation amount. Adds even harmonics at low levels; more complex harmonics when pushed.
Output Drive 0.0 – 1.0 Drive level into the output transformer stage. Final harmonic coloration before signal leaves.
Output Saturation 0.0 – 1.0 Output transformer saturation. Lower than input saturation for typical console behavior.
Low Response −1.0 – +1.0 (±3 dB) Low-frequency response adjustment. Positive values add bass weight (core saturation effect). Model-dependent corner frequency.
High Response −1.0 – +1.0 (±2 dB) High-frequency response. Negative = soft tape-like rolloff; positive = extended HF sparkle. Model-dependent corner frequency.
Compression 0.0 – 1.0 Transformer loading compression effect. Gentle gain reduction at high levels mimicking core saturation-induced compression.

PCH
Punch
Transparent clipper with transient shaper and up to 8x oversampling

Punch is a two-stage loudness processor: a transparent peak clipper followed by transient shaping. The clipper eliminates intersample peaks and creates headroom; the transient shaper restores or enhances attack characteristics that compression may have dulled. With up to 8x oversampling, aliasing artifacts are pushed well beyond the audible range. The pre-clip transient detection ensures the shaping is applied before limiting, preventing the “pumping” artifacts common to post-clip transient processors.

ControlRangeDescription
Clip Mode Hard / Soft / Cubic Hard: abrupt clipping, maximum loudness, adds odd harmonics. Soft: tanh saturation, gentle onset, most transparent. Cubic: polynomial waveshaper, falls between hard and soft — recommended for most buses.
Ceiling −12 – 0 dBFS Peak ceiling. −0.3 to −0.1 dBTP is typical for streaming. −1 dB provides headroom for mp3 intersample peaks.
Oversampling 1x, 2x, 4x, 8x Linear interpolation upsample + IIR downsample. 4x is recommended for quality/CPU balance. 8x for critical mastering work.
Transient Attack 0 – 1.0 Transient enhancement amount. Detects attack envelopes and applies upward expansion to restore punch. Set to taste after compression.
Transient Sustain 0 – 1.0 Sustain portion shaping. Higher values lengthen the sustain envelope of transient events.
Transient Release 0 – 1.0 Speed at which transient detection decays. Affects how quickly the shaper stops enhancing after a transient event.
Mix 0.0 – 1.0 Dry/wet blend for the entire Punch stage. Allows parallel clipping for more transparent loudness maximization.
Transparent Loudness Maximization

For maximum loudness with minimal audible distortion: Mode Soft or Cubic • Ceiling −0.3 dBFS • Oversampling 4x • Transient Attack 0.3–0.5 • Mix 0.8–1.0. Set ceiling first to where the meter just catches peaks; then dial in transient attack to recover kick and snare punch lost to preceding compression stages. The pre-clip transient detection ensures attack restoration happens before limiting, not after — preventing pumping artifacts.

Death Metal Context

Down-tuned guitars (B or lower) create enormous sub-frequency content below 60 Hz that can destroy clarity in a mix. The aggressive HP at 50 Hz and low-mid cut at 280 Hz prevent the guitars from creating a wall of mud while preserving the “chunk” of palm-muted riffs. The Dry/Wet at 0.90 on ButterComp2 welds the double-kick, guitars, and vocals into a single coherent wall — a hallmark of Morbid Angel, Death, and Cannibal Corpse productions. The Pultec LF trick at 60 Hz adds sub bomb without low-mid bloom, keeping the kick cannon-tight. British Transformer at high drive approximates the SSL G-Bus that powered classic Morrisound productions.

Installation

Download Pre-Built Binaries

The recommended path for most users. Download from the latest GitHub release.

Windows

Download Bus-Channel-Strip-windows.zip

VST3: Extract Bus-Channel-Strip.vst3 to:

C:\Program Files\Common Files\VST3\

CLAP: Extract Bus-Channel-Strip.clap to:

C:\Program Files\Common Files\CLAP\

macOS

Download the .tar.gz for your CPU (Intel or Apple Silicon)

VST3:

~/Library/Audio/Plug-Ins/VST3/

CLAP:

~/Library/Audio/Plug-Ins/CLAP/

Linux

Download Bus-Channel-Strip-linux.tar.gz

VST3 (user):

~/.vst3/

VST3 (system):

/usr/lib/vst3/

Build From Source

# 1. Install Rust nightly
rustup toolchain install nightly

# 2. Install LLVM 19+ (Windows — required for Skia)
#    Download from https://github.com/llvm/llvm-project/releases

# 3. Clone and build
git clone https://github.com/YOUR_USERNAME/bus_channel_strip
cd bus_channel_strip

# Windows
set LLVM_HOME=C:\Program Files\LLVM
set LIBCLANG_PATH=C:\Program Files\LLVM\bin
cargo +nightly run --package xtask -- bundle bus_channel_strip --release

# macOS / Linux
cargo +nightly run --package xtask -- bundle bus_channel_strip --release

Bundles are output to target/bundled/.
