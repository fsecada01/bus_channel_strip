# AGENTS

# This document tracks the agents and their roles in the development of the bus channelstrip plugin using NIH-Plug.

## Agents

- **User**: Provides requirements, specifications, and resources for the plugin.
- **Developer**: Implements the plugin code, builds, and tests the plugin.
- **CI**: Automates builds, tests, and validations.
