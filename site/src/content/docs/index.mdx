## Signal Chain

<div class="signal-chain">

Every module is independently bypassable, fully automatable, and reorderable at runtime via the drag-to-swap UI.

## Modules

<div class="module-grid">
  <div class="module-card card-eq">
    <h3>API5500 EQ</h3>
    <p>5-band semi-parametric EQ modeled on the API 5500 console equalizer. HPF, low/high shelf, and two parametric bells.</p>
  </div>
  <div class="module-card card-comp">
    <h3>ButterComp2</h3>
    <p>Airwindows bipolar interleaved compression. Described as the richest glue compressor available. Built-in NY parallel blend.</p>
  </div>
  <div class="module-card card-pultec">
    <h3>Pultec EQ</h3>
    <p>EQP-1A style passive equalizer with simultaneous boost/cut low-end shaping. Includes tube saturation stage.</p>
  </div>
  <div class="module-card card-dyneq">
    <h3>Dynamic EQ</h3>
    <p>4-band frequency-dependent dynamics with Compress, Expand, and Gate modes. Real-time spectral analyzer with GR metering.</p>
  </div>
  <div class="module-card card-xfm">
    <h3>Transformer</h3>
    <p>Vintage/Modern/British/American transformer coloration. Independent input and output stages. Harmonic saturation and frequency shaping.</p>
  </div>
  <div class="module-card card-punch">
    <h3>Punch</h3>
    <p>Transparent clipper with Hard/Soft/Cubic modes, up to 8× oversampling, and pre-clip transient shaper for loudness maximization.</p>
  </div>
</div>
